## Project Description
A complete React-based community platform called CommunityHub. Users can browse posts fetched from an API, create new posts, like posts, search/filter content, and navigate between pages using React Router. Built as part of IYF Academy Week 09 advanced React assignments.

## Technologies Used
- React 18
- React Router v6
- Tailwind CSS
- Custom Hooks (useFetch, useLocalStorage)
- JSONPlaceholder API
- Vite
- JavaScript (ES6+)
- Git & GitHub

## Features
- Home page with welcome banner and recent posts
- Posts list page with search/filter functionality
- Single post detail page
- Create new post form with validation
- Like posts
- Delete posts
- Loading and error states
- Responsive layout with Header, Sidebar and Footer
- Reusable component library (Button, Input, Card, Modal, Avatar)
- React Router navigation with active links

## How to Run
1. Clone this repository
2. Run `npm install`
3. Run `npm run dev`
4. Open browser at `http://localhost:5173`
